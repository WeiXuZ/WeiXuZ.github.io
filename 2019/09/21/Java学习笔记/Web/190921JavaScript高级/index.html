BOM和DOM

BOM：Browser Object Model，浏览器对象模型，是以对象的形式操作浏览器的功能。

DOM：Document Object Model。文档对象模型，是以对象的形式操作HTML标签和CSS样式。

​ 1、获取对象（知道对象的功能）

​ 2、操作对象的属性或方法（对象有哪些属性和方法）

DOM

​ 是什么？能干什么？

​ 就是把HTML中的各种标签，抽象层了很多对象，然后用面向对象的方式来操作这些元素（标签）。

​ 怎么用？

​ 如何获取标签对象：

​ var elementObj = document.getElementById(id属性的值)； // 根据元素的id属性的值去获取一个对象

​ document.getElementsByClassName(类名); // 根据元素的class属性的值去获取一个元素对象数组（伪数组）

​ document.getelementsByTagName(标签名); // 根据元素的标签名去获取一个元素对象数组（伪数组）

​ 标签对象都有哪些属性和方法可以被操作？

​ a: href

​ img: src

​ 操作内容，innerHTML, 操作的是元素开始标签和结束标签中间的内容。

事件及事件驱动

​ 事件的三个要素：

​ 事件源：发生事件的那个东西

​ 事件的处理程序（监听器）：发生事件之后，执行的程序

​ 事件（事件自身）：发生的那件事

​ 事件驱动程序编写的三个步骤：

​ 1、搞一个事件源

​ 2、编写一个事件处理函数（监听器）

​ 3、绑定事件源和事件处理程序
