4. Vue实例生命周期及钩子函数

目标：了解Vue实例生命周期，生命周期的钩子函数及created函数常用场景

分析：

在创建vue实例的时候可以指定模板id、数据和方法；而如果要在实例化、模板渲染的过程中需要执行一些其它操作的话；那么可以使用钩子函数。

小结：

VUE 的声明周期

VUE 的整个执行是有一个动态的过程的,先干什么,后干什么,这个过程我们叫做生命周期

而整个生命周期我们可以定义函数进行干预,vue 的整个过程

vue 自带了一些函数 能够让我们干预,这些函数叫做钩子函数, 只要我们定义好了,VUE底层会自动”钩取”调用

钩子函数会在vue实例的各个生命周期阶段自动调用；具体有：

beforeCreate，

created, (对象创建时)

beforeMount,

mounted, (渲染,就是第一次吧数据赋值到html 元素中)

beforeUpdate，

updated, (更新)

beforeDestroy

destroyed，

钩子函数不要使用箭头函数的方式编写。

7. 指令- v-on使用

目标：了解v-on指令的语法实现按钮点击后的递增和递减

分析：

在没有使用vue之前；页面标签可以通过设置onXXX响应事件；在vue中可以通过v-on指令响应事件。

小结：

v-on的使用：

事件冒泡:

​ 指的是点击顶层标签, 所有标签上的事件自上而下执行,叫冒泡,为啥叫”冒泡”,应为重代码上来看是向上冒泡的

事件修饰符：语法v-on:xxxx.修饰符，常用的修饰符有：

.stop ：阻止事件冒泡 ,(html 组件层叠是只触发最顶层组件的事件)
.prevent ：阻止默认事件发生
.capture ：使用事件捕获模式(VIP)
.self ：只有元素自身触发事件才执行。（冒泡或捕获的都不执行）
.once ：只执行一次
