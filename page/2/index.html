1.添加用户

  • 使用adduser或者useradd添加一个名为ying的用户: adduser ying

  • 修改密码:输入 passwd ying

  • 会看到以下提示,按提示输入两次密码:

    Changing password for user ying.
    New UNIX password: //在这里输入新密码
    Retype new UNIX password: //再次输入新密码

    看到下边这行提示就说明添加成功了:

    passwd: all authentication tokens updated successfully.
